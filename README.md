# CNY 西恩纽约
CNY 西恩纽约 (cnnewyork .com) 是一个基于 TCShop 开发的电商网站。TCShop 基于 ThinkCMFX 面向电商平台，利用 ThinkCMFX 灵活的应用机制，框架自身提供基础的管理功能，而开发者可以根据自身的需求以应用的形式进行扩展。每个应用都能独立的完成自己的任务，也可通过系统调用其他应用进行协同工作。

官网: http://www.cnnewyork.com  
github: https://github.com/MoguTech-Fundation/cnnewyork.com
